### Installation
``` bash
# node.js >= 16.10
corepack enable

# node.js <16.10
npm i -g corepack

# update yarn
yarn set version stable

# init project
yarn init -2
```

``` bash

# list of commands
yarn help

# installing dependencies
yarn install

# installing package
yarn add package@version
yarn add package@tag

# upgrading pkg
yarn up package@version
yarn up package@tag


# removing pkg
yarn remove package
```
